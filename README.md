# RPG para console para IDE (VScode, IntelliJ, Eclipse e NetBeans)
Selecione sua classe (Paladino, Mago, Bardo e Barbaro) para começar a jogar, após isso sera randomizado uma raça (Humano, Elfo, Anão e Goblin), cada raça tem atributos que buffarão os atributos de alguma classe.
